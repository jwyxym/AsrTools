1. **克隆仓库并进入项目目录**

    ```bash
    git clone https://github.com/jwyxym/AsrTools.git
    cd AsrTools
    ```

2. **安装依赖并运行**

    - **需要安装的依赖**

        需要安装的环境变量：[ffmpeg](https://ffmpeg.org/download.html#build-windows)

        ```bash
        pip install -r requirements.txt
        ```

    - **启动 GUI 界面**

        ```bash
        python asr_gui.py
        ```
